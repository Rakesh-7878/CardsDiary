## 📖 About The Project

**CardsDiary** is a state-of-the-art web application designed to help users create, customize, and download stunning digital invitation cards instantly. Built with a focus on modern aesthetics, smooth animations, and a premium user experience, CardsDiary replaces traditional paper invitations with elegant, eco-friendly digital alternatives.

### ✨ Key Features

- 🎨 **Curated Premium Collections:** Beautifully designed templates tailored for:
  - 💍 Weddings
  - 🎂 Birthdays
  - 🥂 Anniversaries
  - 👶 Baby Showers
- ⚡ **Instant Customization & Download:** Personalize text, fonts, and colors, then download instantly in high resolution.
- 📱 **Progressive Web App (PWA):** Installable as a standalone app on mobile and desktop for faster access directly from your home screen.
- 💫 **Interactive UI/UX:** Features a dynamic hero section with floating particles, smooth anchor scrolling, and premium fade-up animations.
- 💬 **Integrated FAQ & Contact:** Seamless modal-based FAQ section and direct WhatsApp/Email integration for support.
- 📱 **Fully Responsive:** Meticulously crafted to look flawless on smartphones, tablets, and desktops.

## 🛠️ Built With

This project is built using modern web fundamentals to ensure blazing-fast performance without heavy framework overhead:

- **HTML5** (Semantic structure)
- **CSS3** (CSS Variables, Flexbox/Grid, Keyframe Animations, Backdrop-filters)
- **Vanilla JavaScript** (ES6+, IntersectionObserver API, HTML5 Canvas, Service Workers)
- **Google Fonts** (Great Vibes, Cormorant Garamond, Montserrat, Poppins)

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser to view the project locally. No build steps or heavy dependencies are required.

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/Rakesh-7878/CardsDiary.git
   ```
2. Navigate to the project directory
   ```sh
   cd CardsDiary
   ```
3. Run a local development server. For example, using Python or VS Code Live Server:
   - Using Python:
     ```sh
     python -m http.server 8000
     ```
   - Using Node.js (http-server):
     ```sh
     npx http-server
     ```
4. Open your browser and navigate to `http://localhost:8000`

CardsDiary is fully PWA-ready! Users visiting the site will be prompted to "Download the App" to install it directly to their device, providing a native app-like experience with an immersive, full-screen UI.
